# HOME主要グラフ

HOMEのグラフは、既存のHOMEカード・店舗内訳で利用している期間、店舗範囲、CTI/Town/Heavenのデータを表示用に再構成したものです。新しい保存値や集計定義は追加していません。

## グラフ

- 累積売上・目標進捗：日別売上を累積し、設定済み月目標を日数按分。予測は既存カードと同じ単純ペース式。
- 日別売上・出勤人数：棒が売上、線が同日重複を除いたCTI出勤人数。出勤時間・成約などはツールチップで確認。
- 店舗別比較：売上、報酬、成約、出勤、効率、Town、Heavenを切替。媒体データなしは棒を描かず「—」。
- キャスト効率分布：CTIアクティブ、出勤1日以上、出勤時間>0のみ。LOW_SAMPLEは灰色で表示し、点からCAST DISCOVERYへ移動。

未来日の実績は補完せず、未存在媒体を0に置換しません。HOMEカードと同じサーバー側データから生成します。
